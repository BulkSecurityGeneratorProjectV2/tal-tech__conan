# conan

柯南流量回放整体服务 
http://conan.xesv5.com

## 一、 背景

柯南是一站式的开源流量回放平台，基于线上流量回放能力助力开发和测试团队快速拥有相关能力，加速高质量软件的交付。

-   流量采集: ES日志；
-   回放协议: http协议；
-   接入成本: 无需安装采集，对服务性能无影响；
-   支持场景: 目前支持读接口。

## 二、项目介绍

### 1. 工程文件目录

柯南项目整体采用的父子结构

conan-common存放的是公共资源

conan-admin提供的是与前端数据交互的接口

conan-agent提供的是比较费时的服务包括录制回放和比对
